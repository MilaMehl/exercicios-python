# exercicios-python
Este repositório contém exercícios práticos de Python que estou desenvolvendo durante meu curso online de aprendizado próprio. 

## Estrutura do repositório
- 'modulo1/' -> Exercícios básicos do inicio do curso
- 'modulo2/' -> exercicios intermediários a avançados do final do curso

## Como executar os exercícios
Certifique-se de ter o **Python 3** instalado em sua máquina.  
Para verificar, rode no terminal:
   python3 --version
Navegue até a pasta do exercício desejado e execute o arquivo .py correspondente.
