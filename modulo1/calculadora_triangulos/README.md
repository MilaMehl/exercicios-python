# Calculadora de Hipotenusa em Python

Um script simples para aplicar o Teorema de Pitágoras e calcular a hipotenusa de qualquer triângulo retângulo.

# Funcionalidades

* Coleta o cateto oposto e o cateto adjacente.
* Utiliza o módulo `math` para cálculo da raiz quadrada.
* Exibe o resultado da hipotenusa no terminal.

# Instalação e uso

Certifique-se de ter o Python 3 instalado na sua máquina. Você pode verificar isso executando o seguinte comando no seu terminal:
   python --version

1. Baixe o arquivo "calculadora_hipotenusa"
2. Abra o terminal na pasta do arquivo baixado.
3. Execute o seguinte comando: 
   python calculadora_hipotenusa.py
4. Siga as instruções no terminal.
